# Kitty Genomes (kitty-genomes)
kitty database kitty analysis

## Getting Started
##### Create `credentials.json` file in the root folder and add your jwt authorization in this format.
```javascript
{
  "url": "https://api.cryptokitties.co/",
  "credentials": {
    "jwt": <jwt>
  }
}
```
##### Run `npm install`
##### Start MongoDB Server
##### Run `npm start`
